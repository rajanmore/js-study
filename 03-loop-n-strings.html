03 Loop and strings

Loops can execute a block of code a number of time

Kinds of Loops

  1. for loop: Loops through a block of code a number of times
  2. while loop: Loops through a block of code until a certain condition is met
  3. do while loop: Loops through a block of code until a certain condition is met
  4. for-of loop: Loops through a block of code until a certain condition is met
  5. for-in loop: Loops through a block of code until a certain condition is met
